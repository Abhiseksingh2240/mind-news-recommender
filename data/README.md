# mind-news-recommender
Implementation and evaluation of baseline, collaborative, content-based, and hybrid recommendation methods on the MIND news dataset.

# Dataset Instructions

This project uses the MIND dataset released by Microsoft.

1. Download the dataset from the official source:
   https://msnews.github.io/

2. Choose the MIND-small or MIND-large version.

3. Extract the files into the following directory:
   data/raw/

Expected files:
- news.tsv
- behaviors.tsv
